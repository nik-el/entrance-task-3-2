# Запуск

```
node script.js
```
## Функционал
+ На входе принимает список устройств и тарифы и максимальную мощность
+ На выходе выдает расписание и затраченную энергию по устройству
+ Следит за тем, чтобы не была превышена максимальная мощность в час
